Theme Configuration

Cleopatra uses CSS variables for theming, inspired by shadcn/ui. Customize your theme by editing src/styles/theme.css

Color Variables

--primary
--secondary
--destructive
--success
--warning
--info

How to Use

/* In your CSS */
.my-button {
  background: hsl(var(--primary));
  color: hsl(var(--primary-foreground));
  border-radius: var(--radius);
}

/* Or use utility classes */
.my-element {
  @apply bg-primary text-primary-foreground;
}
