Features

Everything you need.

From authentication flows to data visualization, Cleopatra provides the building blocks you need to ship production-ready admin dashboards.

Themable

Customize with CSS variables. Light and dark mode included. Make it yours with minimal effort.

Responsive

Mobile-first design ensures your dashboard looks great on any device, from phones to ultrawide monitors.

Lightweight

No heavy frameworks. Pure HTML, CSS, and vanilla JS. Blazing fast performance out of the box.

Accessible

Built with WAI-ARIA guidelines in mind. Keyboard navigation, screen reader support, and focus management.

Developer First

Clean, readable code with sensible conventions. No black-box magic—understand everything.

Open Source

MIT licensed. Use it for personal and commercial projects. Contribute and shape the future.
